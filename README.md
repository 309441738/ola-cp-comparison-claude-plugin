# Cruise Product Comparison · Claude Code 插件

在选择邮轮时，您可以比较不同邮轮产品的价格、行程和品牌，以帮助您做出更明智的决策。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-cp-comparison-claude-plugin
/plugin install ola-cp-comparison@ola-cp-comparison-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Cruise Product Comparison**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
